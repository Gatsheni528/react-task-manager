# 📝 React Task Manager App

A responsive, theme-aware task manager and blog viewer built using **React**, **Vite**, **Tailwind CSS**, and **JSONPlaceholder API**.

## 📂 Features

### ✅ Task Manager
- Add, complete, delete tasks
- Filter: All / Active / Completed
- Tasks saved in browser using `localStorage`
- Clean UI with responsive layout

### 🌐 Blog Post Viewer
- Fetches posts from JSONPlaceholder API
- Search functionality
- Pagination / Load more button
- Loading and error handling

### 🎨 Theming & Styling
- Light / Dark Mode toggle using `useContext`
- Tailwind CSS utility-first styling
- Responsive design for mobile, tablet, and desktop
- Transitions and hover effects for better UX
